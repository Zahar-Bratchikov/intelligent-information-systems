# Распознавание геометрических фигур с помощью нейросети

## Архитектура нейронной сети
- Входной слой: 64 нейронов (8×8 пикселей)
- Скрытый слой: 32 нейрона, функция активации — сигмоида
- Выходной слой: 8 нейронов (по числу фигур), функция активации — сигмоида
- Обучение: метод обратного распространения ошибки (backpropagation)
- Алгоритм: стохастический градиентный спуск

## Фигуры
1. Круг
2. Квадрат
3. Треугольник
4. Прямоугольник
5. Пятиугольник
6. Шестиугольник
7. Звезда
8. Ромб

## Управление
- Рисуйте мышью на холсте (8×8 сетка)
- Добавляйте/удаляйте/редактируйте образцы
- Обучайте сеть и распознавайте фигуры
